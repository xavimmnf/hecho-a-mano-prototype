# Hecho a Mano

## Propósito

Hecho a Mano es una plataforma web que conecta artesanos con personas que valoran productos hechos a mano. Permite explorar catálogos, conocer historias y perfiles de artesanos, ver detalles de productos y gestionar la compra desde un flujo básico de carrito y checkout.

## Instalación

1. Clona o descarga el repositorio.
2. Instala dependencias:
   ```bash
   npm install o npm i
   ```

## Uso

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```

Para verificar calidad del código:

```bash
npm run lint
```

Para construir el proyecto (producción):

```bash
npm run build
```

Visualizar la versión construida:

```bash
npm run preview
```

## Tecnologías utilizadas

- React
- React Router
- Vite
- ESLint
- CSS (incluye estilos por componentes/páginas)
- styled-components
- TailwindCSS (config presente en dependencias)
- lucide-react / react-icons (iconografía)
